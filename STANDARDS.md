# Information Classification Standards

## 1. 核心层 (Tier 1: Core-Decision)
**定义**: 直接影响未来 3 个月生存、安全、重大利益的信息。
- **范畴**: 极端天气预警、重大政策变动（个税/医保/房产）、行业巨变、家庭紧急事件。
- **处理要求**: 
  - **即时响应**: 看到后 1 小时内转化为行动（Action）。
  - **阅后即焚**: 行动完成后，信息可归档，不占用长期注意力。

## 2. 认知层 (Tier 2: Cognitive-Framework)
**定义**: 塑造世界观、理解运行规律，但不直接影响当下决策。
- **范畴**: 国际地缘政治趋势、宏观经济分析、科学突破、社会思潮演变。
- **处理要求**: 
  - **深度关联**: 必须找到 1 个与现有知识库的关联点。
  - **批判思考**: 记录不同立场的视角对比。

## 3. 社交层 (Tier 3: Social-Connection)
**定义**: 用于日常社交、情感共鸣或调节情绪的信息。
- **范畴**: 邻里动态、大众热点、娱乐/体育八卦。
- **处理要求**: 
  - **极简摘要**: 仅需了解基本事实。
  - **定期清理**: 超过 7 天未产生价值的信息直接淘汰。

## 4. 专业层 (Tier 4: Professional)
**定义**: 针对具体问题或技能提升的信息。
- **范畴**: 技术文档、行业深度报告、工具教程。
- **处理要求**: 
  - **问题导向**: 只有在需要解决具体问题时才调取并深入学习。
  - **系统沉淀**: 转化为永久笔记存入个人知识库（Zettelkasten）。
